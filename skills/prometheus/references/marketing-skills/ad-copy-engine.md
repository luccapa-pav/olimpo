# Ad Copy Engine — Long, Short e UGC Script

Produz 3 formatos simultaneamente: long-form (Meta), short-form (Google), UGC script (vídeo).

## 3 Formatos por conceito

### Format 1: Long-Form (Meta Primary Text, 100-250 palavras)
- Hook (primeira linha para o scroll)
- Pain amplification (2-3 frases)
- Pivot pra solução (1 frase)
- Credibility signal (prova, número, testemunho)
- Feature-to-benefit stack (3-5 bullets: feature → outcome)
- Objection handling (razão #1 pra NÃO clicar)
- CTA claro e específico

### Format 2: Short-Form (Google RSA)
- 5 headlines (max 30 chars cada)
- 3 descriptions (max 90 chars cada)
- Funcionam em qualquer combinação

### Format 3: UGC Script (30-60 segundos falados)
- Hook (3s — pattern interrupt ou bold claim)
- Problem acknowledgment (5-8s — "sabe quando...")
- Discovery moment (5s — "encontrei / alguém me mostrou")
- Product reveal + benefício (10s)
- Proof/resultado (5-8s)
- CTA (3-5s — conversacional, não vendedor)

## Regras UGC
- Linguagem falada, não escrita
- Usar contrações
- 1-2 filler words (tipo, então, literalmente)
- Uma auto-correção por script ("não, peraí, o que quero dizer é...")
- Se soa como anúncio, reescrever

## Regras gerais
- Sempre carregar brand voice
- UMA mensagem-chave por peça
- 3 conceitos por pedido
