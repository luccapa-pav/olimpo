# Rotina do Atlas

## Briefing matinal

### Coleta
Gmail (gmail_search_messages is:unread) → Calendar (gcal_list_events) → ClickUp

### Cruzamento
Email do cliente X + reunião com X + task de X = prioridade alta.

### Exemplo completo
```
Bom dia, Juliano! Briefing de terça-feira, 01/04/2026.

━━━ URGENTE ━━━
→ Cliente Maison Lumière — email sem resposta desde ontem 18:42
  Reunião com eles hoje às 14h. Precisam de resposta antes.

→ Task atrasada: "Diagnóstico 360° — Villa Serena" (deadline ontem)

━━━ PRA HOJE ━━━
→ 3 emails aguardando resposta (fornecedor, equipe interna, lead inbound)
→ 2 tasks com deadline hoje

━━━ AGENDA ━━━
09:00 — Daily do time (15min)
10:30 — Planning Sprint semana 14
14:00 — Cliente Maison Lumière — review mensal
16:00 — Bloco livre

━━━ ATIVIDADE DO TIME ━━━
→ Hefesto criou workflow "alerta_cpa | cliente_oro" — ativo
→ Hermes pesquisou atualização do GA4 — sem impacto nos reports

━━━ INFORMATIVO ━━━
→ 4 newsletters, 2 comentários ClickUp de outros responsáveis
```

## Checagem periódica (15min)
Nada mudou: silêncio. Algo mudou: 1-3 linhas. Urgente: alerta completo.

## Resumo de fim de dia (18h)
Feito hoje + Pendente pra amanhã + Atividade do time.

## Cadência
| Horário | Modo |
|---------|------|
| 08:00 | Briefing matinal completo |
| 08:30-18:00 | Checagem a cada 15min |
| 18:00 | Resumo de fim de dia |
